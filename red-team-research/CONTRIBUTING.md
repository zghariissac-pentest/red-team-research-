# Contributing

## Requirements

- Technical accuracy
- Clear documentation
- Ethical and legal compliance
- Proper references

## Markdown Standards

- One topic per file
- Use headings consistently
- Include references where possible

## Pull Request Process

1. Fork repository
2. Create branch
3. Add changes
4. Submit pull request

## Prohibited Content

- Malware source code
- Unauthorized attack instructions
- Illegal activities
- Stolen data
