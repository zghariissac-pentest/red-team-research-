# Courses Recommended

## Overview

This file contains a structured list of recommended courses for building strong skills in cybersecurity, with a focus on offensive security, red teaming, and practical hands-on learning.

---

# 1 Foundations

## Networking Fundamentals

Recommended courses:
- Computer Networking basics (TCP/IP, DNS, HTTP)
- Network protocols and architecture
- Practical packet analysis introduction

## Operating Systems

Recommended courses:
- Linux fundamentals and administration
- Windows internals basics
- System processes and memory concepts

## Programming Basics

Recommended courses:
- Python for automation and scripting
- Bash scripting for Linux environments
- Basic C for low-level understanding
