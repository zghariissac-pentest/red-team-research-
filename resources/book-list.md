# Book List

## Overview

This directory contains a curated collection of books for offensive security, red teaming, penetration testing, system administration, networking, malware analysis, cloud security, and related disciplines.

The purpose of this collection is to provide a structured learning path from foundational knowledge to advanced red team operations.

---

## Learning Philosophy

Red teaming is a multidisciplinary field. Effective operators typically develop knowledge in:

* Networking
* Operating Systems
* Programming
* Web Technologies
* Active Directory
* Cloud Infrastructure
* Security Testing
* Detection Engineering
* Threat Intelligence
* Malware Analysis

This collection is organized to reflect that progression.

---

## Recommended Learning Order

### Stage 1 Foundations

Before studying offensive security, build strong fundamentals in:

* Computer Networks
* Linux Administration
* Windows Internals
* Programming
* Web Technologies

### Stage 2 Security Fundamentals

Focus on:

* Security Concepts
* Defensive Security
* Threat Modeling
* Security Architecture

### Stage 3 Offensive Security

Study:

* Penetration Testing
* Web Application Security
* Active Directory Security
* Network Security Assessments

### Stage 4 Advanced Topics

Explore:

* Red Team Operations
* Adversary Emulation
* Malware Analysis
* Reverse Engineering
* Cloud Security

### Stage 5 Research and Specialization

Specialize in:

* Threat Intelligence
* Detection Engineering
* Advanced Tradecraft
* Security Research

## Study Recommendations

1. Read actively and take notes.
2. Build practical labs alongside reading.
3. Document findings and lessons learned.
4. Revisit foundational topics regularly.
5. Focus on understanding concepts before tools.
6. Validate theory through hands-on experimentation.
